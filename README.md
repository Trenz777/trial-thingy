# trial-thingy
Cómo funciona: Usando la ecuacion Largo contenedor / largo rectangulo * ancho contenedor / ancho rectangulo se puede calcular un aproximado de cuantos caben, mejor si se compara el maximo entre esa y largo cont / ancho rect * ancho cont / largo rect para comprobar si rotando caben más. Existe una comprobación para ver si la dimensión mas pequeña del contenedor alcanza para los rectángulos también, si no, devuelve 0.
